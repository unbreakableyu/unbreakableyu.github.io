解题思路

大体与CS50-Pset2-Caesar的思路相似：

Get key -> Validate key -> Get Plaintext -> Encipher -> Print Ciphertext

Validate key重点

检查是否有重复的字母，此处最开始设想依次比较的方式，但直觉太麻烦且会把本就是糨糊状态的大脑弄得更混沌，于是想着使用Scrabble里的方法：

给每一个字母赋予一个特殊的值(刁钻地选了一些质数)，计算总和，如果总和不等于某个确定的值，就说明其中有重复。

Encipher重点

其中Encipher的思路：ASCII -> alpha-index maps to key-index -> ASCII

plaintext[i] isalpha?
N -> ciphertext[i] = plaintext[i]
Y -> get alpha-index
isupper? Y -> key-index + 65; N -> key-index + 97
